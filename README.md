# bug_report3.21
This application will allow users to report any bugs or typos that they encounter while surfing the web. 
The reports will be attached to the submitting user and will eventually allow the user to upload a screenshot of the issues. 
One of the features of this particular tool is that it allows users of all technical skill levels to suggest fixes. 

Technologies and Tools:
Python 3.9.0
Flask 1.1.2
Werkzeug 1.0.1
SQLAlchemy-1.3.23 flask-sqlalchemy-2.4.4 (model manipulation)
flask-login-0.5.0 (manages user login/out)
Bootstrap
